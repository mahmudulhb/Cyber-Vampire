# Sensitive Data Redaction (Flask App)

This project is a simple web-based tool that can detect and redact sensitive information such as:
- Names
- Locations
- Dates
- Times
- Emails
- IP addresses
- Phone numbers
- Credit card numbers
- URLs
- SSN
- Custom regex-based entities

It uses:
- Python (Flask)
- spaCy NLP (en_core_web_lg)
- Custom Regex Patterns
- HTML frontend

###  How to Use
1. Open the web page
2. Paste any text
3. Select **Mask** or **Remove**
4. Click **Redact**
5. View:
   - Redacted text
   - Detected entities (with labels)

---

###  Troubleshooting

**spaCy model error:**  
Run:
python -m spacy download en_core_web_lg

**Flask error:**  
Check Python version (must be 3.9+).

**Regex not working:**  
Make sure your app.py file contains updated REGEX_PATTERNS.
